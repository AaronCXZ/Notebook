#### 4种常见地二分查找变形问题
1. 查找第一个值等于给定值的元素
- mid == 0 || a[mid-1] != v
- 当a[mid] == v时，mid == 0时是第一个元素，a[mid -1] != v时是第一个等于v的元素 
2. 查找最后一个值等于给定值的元素
- mid == n-1 || a[mid+1] != v
- mid == n - 1时是最后一个元素 ，a[mid +1] != v时是最后一个等于v的元素
3. 查找第一个大于等于给定值的元素
- mid != n-1 && a[mid+1] > v
4. 查找最后一个小于等于给定值的元素
- mid == 0 || a[mid-1] < v
